Figure shows how to verify an A.C.L.  The first command is run in privilege exec mode and it is, show access hyphen lists.  This lists the access lists on the router along with the lines in each one.  The two access lists created in the previous page are listed.  The second command is, show i.p. interface gig zero slash zero.  This lists the access lists that are configured on the interface along with the direction.  The figure shows that the outgoing list is browsing and the inbound list is surfing.
