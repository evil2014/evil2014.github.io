Figure shows the process for using a text editor to edit a numbered A.C.L.  The first command creates a numbered A.C.L. with the number 1. The A.C.L. can be viewed using the command show running hyphen config pipe include access hyphen list 1.  The A.C.L. can be copied and pasted into a text editor and edited.  The current A.C.L. is removed from the interface using the keyword no.  The new A.C.L. is then pasted into the router and can be verified with the command, show running hyphen config pipe include access hyphen list 1.
