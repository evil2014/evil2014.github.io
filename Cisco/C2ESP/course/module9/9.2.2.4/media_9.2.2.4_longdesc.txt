Figure one shows the use of the command, show i.p. interface, to display any A.C.L.s applied.  The figure shows the output from the command, show i.p. interface serial zero slash zero slash zero. The following line is highlighted, outgoing access list is 1.  The command is run again for the gigabit Ethernet interface and the output lists outgoing access list is no underscore access list, which is a named A.C.L. Figure two shows the output from the command, show access hyphen lists.  This will display the full contents of all A.C.L’s on a router, but doesn’t specify where each A.C.L. is placed.
