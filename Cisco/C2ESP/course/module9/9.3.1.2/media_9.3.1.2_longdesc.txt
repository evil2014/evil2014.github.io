Figure one shows an example of an extended A.C.L. using port numbers.  The figure lists three configuration lines. The first line is, access hyphen list 114 permit t.c.p. 192 dot 168 dot 20 dot zero zero dot zero dot zero dot 255 any e.q. 23.  This will block telnet traffic.  The next two lines will block f.t.p. traffic. The lines are, access hyphen list 114 permit t.c.p. 192 dot 168 dot 20 dot zero zero dot zero dot zero dot 255 any e.q. 21, and access hyphen list 114 permit t.c.p. 192 dot 168 dot 20 dot zero zero dot zero dot zero dot 255 any e.q. 20.  The figure also lists the alternate method using keywords in place of port numbers.  Using this method the three lines would read, access hyphen list 114 permit t.c.p. 192 dot 168 dot 20 dot zero zero dot zero dot zero dot 255 any e.q. telnet, access hyphen list 114 permit t.c.p. 192 dot 168 dot 20 dot zero zero dot zero dot zero dot 255 any e.q. f.t.p., and access hyphen list 114 permit t.c.p. 192 dot 168 dot 20 dot zero zero dot zero dot zero dot 255 any e.q. f.t.p. hyphen data. Figure two shows the use of the question mark character at the C.L.I. in order to get a listing of port numbers.  This list is displayed by using the command, access hyphen list 101 permit t.c.p. any any e.q. question mark.
