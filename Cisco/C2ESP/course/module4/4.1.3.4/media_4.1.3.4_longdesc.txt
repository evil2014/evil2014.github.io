Figure shows the commands used to assign an i.p. address on a loopback interface on a router.  The commands in the figure begin from global configuration mode.  The first command moves to interface configuration mode, and it is interface loopback zero.  The next command in the figure assigns an I.P. address and subnet mask on the interface, and it is i.p. address 10 dot 0 dot 0 dot 1 255 dot 255 dot 255 dot 0.
