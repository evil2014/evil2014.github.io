Figure one shows the commands necessary to configure a specified link local address on an interface.  The first command is, i.p.v.6. address f e 8 0 colon colon 1 link hyphen local. This command configures the address for the gigabit Ethernet interface.  The figure applies the same address to both of the serial interfaces using the same command.  Figure two shows the output from the command, show i.p.v.6. interface brief, and highlights the link local address on each interface. Figure three is an interactive media element that allows the learner to configure link local addresses on router R-2.
