Figure one shows the output from the command, show port hyphen security interface fast Ethernet zero slash 18.  The setting for maximum MAC addresses is set to 1 in this example and is highlighted. Figure two highlights the fact that the new maximum number is set to 50, and that sticky MAC addresses is now set to one. Figure three shows the output from the command, show run pipe begin fast Ethernet zero slash 19.  The output lists the configuration of the port, including the port security.  A MAC address is listed in the configuration and represents a computer being connected to the switch port and being added to the running configuration file. Figure four shows the output from the command, show port-security address.  The figure lists the MAC address of the computer that was added to the switch in the previous figure as a secure sticky MAC address.
