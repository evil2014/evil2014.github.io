Figure one shows the I.P.V.6. routing table for router R-1 produced from the command, show i.p. route.  The table consists of three directly connected routes, three interface I.P. addresses, and three networks learned from E.I.G.R.P. Figure two highlights the three directly connected networks and the three interface I.P. addresses. Figure three lists one of the entries and identifies each piece of the entry.  The line reads C for a directly connected route, 2 0 0 1 colon D B 8 colon C A F E colon A 0 0 1 colon colon slash 64 for the network address, 0 slash 0 for the administrative distance and the metric.  The line ends with via serial zero slash zero slash zero for the outgoing interface.
