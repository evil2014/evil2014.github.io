Figure shows two routers connecting to a WAN cloud using P.P.P.  The figure shows the output from the configuration file, and is highlighting the line that reads p.p.p. quality eighty.  The command used to configure the quality is p.p.p quality.  This command is followed by a value between one and one hundred. Figure two is an interactive activity that allows the learner to configure a serial interface with an i.p. version 4 address, an i.p. version 6 address, p.p.p. encapsulation, compression, and a quality percentage.
