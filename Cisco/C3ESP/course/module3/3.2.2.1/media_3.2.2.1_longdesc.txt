Figure one shows the output from the command, show interface port hyphen channel 1.  This command displays status information regarding the ether channel. Figure two shows the output from the command, show ether channel summary.  This command displays one line of information per port channel when several port channel interfaces are configured on the same device. Figure three shows the output from the command, show ether channel port hyphen channel.  This command displays information about a specific port channel interface. The figure highlights the protocol and the ports configured in the specified port channel. Figure four shows the output from the command, show interfaces f zero slash one ether channel.  This command displays information about the role of the specified interface in the Ether Channel. The figure is highlighting various ether channel information, such as the port state and the partner port information.
