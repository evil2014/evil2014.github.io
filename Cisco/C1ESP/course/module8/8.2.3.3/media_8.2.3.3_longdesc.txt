The figure shows the six types of I.P. v-6 unicast addresses with a few examples.  The address types are Global Unicast, link-local, the loopback addreress, which is colon colon 1 slash one hundred twenty-eight, the unspecified address, which is colon colon slash one hundred twenty-eight, a link local which will be in the range of F C zero zero colon colon slash seven through F D F F colon colon slash seven.  The last type of address is the embedded I.P. v-4 address.
