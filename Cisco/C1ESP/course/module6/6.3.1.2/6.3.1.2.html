Routers

Anatomía de un router

Al igual que las computadoras, las tablet PC y los dispositivos inteligentes, los dispositivos Cisco requieren una CPU para ejecutar las instrucciones del OS, como la inicialización del sistema y las funciones de enrutamiento y conmutación.

La CPU requiere un OS para ofrecer funciones de enrutamiento y conmutación. El Sistema operativo Internetwork (IOS, Internetwork Operating System) de Cisco es el software de sistema usado para la mayoría de los dispositivos Cisco, independientemente del tamaño y el tipo de dispositivo. Se usa en routers, switches LAN, pequeños puntos de acceso inalámbrico, grandes routers con decenas de interfaces y muchos otros dispositivos.

El componente destacado en la ilustración es la CPU de un router Cisco 1941 con disipador térmico acoplado.
